Hybrid

Concepto: Es una manera de utilizar tecnologias web en devices como si se tratase de una applicacion nativa.

Tecnicamente: Es el browser stock del device interpretando la web app (Single Page Application generalmente) y a la vez intefazeando con el hard dandonos acceso al devices, por ejemplo, usar la camara, leer sus contactos, etc..

Firmas y certificados

Android:

Certificado

			
				keytool -genkey -v
				-alias androidreleasekey -keyalg RSA
				-keysize 2048 -validity 14000
				-keystore release.keystore
				-keypass %KEYPASS%
				-storepass %STOREPASS%

Firma

			
				jarsigner -verbose -digestalg SHA1
				"%FILENAME%-release-unsigned.apk" 
				 androidreleasekey
				-sigalg MD5withRSA
				-keystore release.keystore
				-keypass %KEYPASS%
				-storepass %STOREPASS%

Verificacion

			
				jarsigner -verify
				"%FILENAME%-release-unsigned.apk"
				-keystore release.keystore

Zipalign

			
				zipalign -v 4
				"%FILENAME%-release-unsigned.apk" 
				"%FILENAME%.apk"
